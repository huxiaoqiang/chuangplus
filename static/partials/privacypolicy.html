隐私条款

一. 本网站尽一切合理的努力，保护您的隐私信息不被泄露给无关的第三方，保守您的秘密。如果您有任何疑问，请联系客服。

二. 您可以随时访问在注册时被要求填写的信息，随时修改这些信息，包括信息的公开或保密状态

三. 隐私信息包括：

1. 您的住址；

2. 您的联系方式；

3. 您的账号、密码；

4. 您不希望公开的其他信息；

四. 下列信息不属于隐私信息：

1. 您发布的项目信息

2. 您希望公开的其他信息

3. 您在本网站上进行登录、操作等活动过程中本网站记录的相关信息；

4. 为了其他用户的利益必须公开的其他信息

五. 下列情况下，本网站有权公布或向第三方披露您的有关信息：

1. 您以明确的方式授权

2. 法律法规需求

3. 仲裁机构的裁决的要求

4. 司法机关的判决、裁定、协助执行通知书或者其他法律文件要求

5. 行政机关要求

6. 您违反了使用条款的规定或者有其他损害创加权益的行为；

7. 向您提供您需要的信息，必须公布或向第三人披露您的有关信息；

8. 为了您的利益而必须公布或向第三人披露您的有关信息的其他情形

六. 下列情况视为您"以明确的方式授权"，本网站向第三人披露您的有关信息：

1. 在本网张交换名片

2. 在本网站对其他项目信息进行了回应

3. 在本网站发布信息；

4. 在本网站填写信息时，勾选了"公开"或者其他类似的选项；

5. 您以信函、电子邮件或传真等有形的方式授权；

6. 在必要情况下您以口头授权，后以信函、电子邮件或者传真等有形的方式确认此项授权；或者本网站得到您的口头授权后，以信函、电子邮件或者传真等有形的方式向您确认是否授权，授权信息的范围，内容等，您在半小时内未予答复，默认了此项授权。

7. 其他明确的方式。

七. 下列情形导致您的隐私信息被泄露，本网站不承担任何责任：

1. 不可抗力；

2. 意外事件

3. 本网站采取了合理的防护措施而遭到黑客攻击、电脑病毒侵入；

4. 您未能妥善保管您的隐私信息

5. 您主动向第三人透露您的隐私信息；

6. 其他不可归责于本网站的原因。
